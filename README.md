# ai-diagnosis-service5

AIキャリア診断サービス「5問でわかる、あなたのキャリア」のランディングページ（LP）。

5問の質問に答えるだけで、AIがあなたに最適なキャリアロードマップを提案するWebサービスのLPです。

## 技術スタック

- Next.js 15 (App Router)
- React 19
- TypeScript
- MUI (Material UI)

## 主な構成

- ヒーローセクション（動画的な宇宙背景）
- 特徴セクション（3パネル / 星座ビジュアル）
- 診断ステップ（ネオンカード3枚）
- CTAセクション（ロケット打ち上げの動画的背景）
- フッター

## セットアップ

```bash
npm install
npm run dev
```

開発サーバーは http://localhost:3000 で起動します。

## ビルド

```bash
npm run build
```
