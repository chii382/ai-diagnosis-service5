# ユーザージャーニーマップ

## 対象範囲
本ドキュメントでは、Stage1（LP=トップページのみ）における **「LP訪問 → 内容理解 → CTA押下」** のジャーニーを定義する。

- **対象ユーザー**: [01_ペルソナ定義](./01_ペルソナ定義.md) の3タイプ（就活生 / 転職検討 / キャリア迷子）
- **ゴール（KGI）**: CTAボタン（無料診断を始める）の押下
- **対象画面**: トップページ（LP）のみ

---

## LPの構成要素（前提）
1. ヒーローセクション（キャッチコピー、サブコピー、CTAボタン）
2. サービス特徴3つ
3. 診断の流れ（3ステップ）
4. CTAボタン（ページ下部）
5. フッター

---

## ジャーニー全体像（フェーズ）

| フェーズ | ① 訪問 / 第一印象 | ② 内容理解 / 興味喚起 | ③ 納得 / 不安解消 | ④ 行動（CTA押下） |
|----------|------------------|----------------------|------------------|-------------------|
| 対応セクション | ヒーロー | サービス特徴3つ | 診断の流れ3ステップ | 下部CTA / フッター |
| ユーザー状態 | 流入直後 | 「自分に関係ある?」 | 「やってみる価値ある?」 | 「やってみよう」 |

---

## フェーズ別ジャーニーマップ

### ① 訪問 / 第一印象（ヒーローセクション）

| 項目 | 内容 |
|------|------|
| ユーザー行動 | SNS広告・検索・紹介などからLPに流入。ファーストビューを数秒で判断 |
| 思考・感情 | 「これは何のサービス?」「自分に関係ある?」（興味と警戒が半々） |
| タッチポイント | キャッチコピー、サブコピー、メインビジュアル、ヒーローCTA |
| ペインポイント | 何のサービスか一目で分からないと即離脱する |
| 提供価値 / 施策 | 「5問でわかる、あなたのキャリア」で価値を即時提示。3分で完了の手軽さを明示 |
| 感情曲線 | 😐 → 🙂（期待） |

### ② 内容理解 / 興味喚起（サービス特徴3つ）

| 項目 | 内容 |
|------|------|
| ユーザー行動 | スクロールして特徴を読む |
| 思考・感情 | 「他の自己分析と何が違う?」「自分の悩みに効く?」 |
| タッチポイント | 特徴カード3枚（例：①5問で簡単 ②AIが診断 ③ロードマップが手に入る） |
| ペインポイント | 抽象的すぎると「結局何ができるの?」と離脱 |
| 提供価値 / 施策 | アイコン+短文で「手軽さ」「客観性」「具体的な成果」を端的に伝える |
| 感情曲線 | 🙂 → 😊（共感） |

### ③ 納得 / 不安解消（診断の流れ3ステップ）

| 項目 | 内容 |
|------|------|
| ユーザー行動 | 利用イメージを確認する |
| 思考・感情 | 「難しくない?」「時間かからない?」「無料?」 |
| タッチポイント | ステップ図（例：①質問に答える → ②AIが分析 → ③結果を受け取る） |
| ペインポイント | 手間・時間・費用への不安が残ると行動しない |
| 提供価値 / 施策 | 3ステップで「簡単・短時間」を可視化。所要時間や無料であることを明示 |
| 感情曲線 | 😊 → 😄（安心・納得） |

### ④ 行動 / CTA押下（下部CTA・フッター）

| 項目 | 内容 |
|------|------|
| ユーザー行動 | 下部CTAボタンを押下し、診断へ進む |
| 思考・感情 | 「これならやってみよう」 |
| タッチポイント | 下部CTAボタン、フッター（補足情報・信頼要素） |
| ペインポイント | CTAが目立たない / 文言が弱いと押されない |
| 提供価値 / 施策 | 行動を後押しする文言（例：「無料で診断を始める」）を明確なボタンで配置 |
| 感情曲線 | 😄 → ✅（コンバージョン） |

## 感情曲線（テキストイメージ）

```
感情
😄              ┌──────●(納得)──────●(行動)
😊        ┌─────●(共感)
🙂   ┌────●(期待)
😐 ●(流入)
   └──────────────────────────────────→ 時間
   ①ヒーロー  ②特徴   ③ステップ   ④CTA
```

---

## 離脱ポイントと対策

| 離脱が起きやすい箇所 | 主な原因 | 対策（LP設計への反映） |
|----------------------|----------|------------------------|
| ファーストビュー | 価値が伝わらない | キャッチ・サブコピーで一瞬で価値提示 |
| 特徴セクション | 自分ごと化できない | ペルソナの悩みに直結する言葉を使う |
| 流れセクション | 手間・不安が残る | 「5問・3分・無料」を明示 |
| CTA | ボタンが弱い | 明確で目立つCTA、複数箇所に配置 |

## 成功指標（参考・Stage1）

| 指標 | 説明 |
|------|------|
| CTAクリック率（CTR） | LP訪問者のうちCTAを押した割合（主要指標） |
| スクロール到達率 | 各セクションまで到達した割合 |
| 直帰率 | ファーストビューで離脱した割合 |

> **示唆**: 最重要KPIはCTAクリック率。各フェーズの感情をスムーズに次へつなぎ、不安を残さず④の行動へ導く設計が必要。
