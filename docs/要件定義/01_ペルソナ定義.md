# ペルソナ定義

## サービス概要
**「5問でわかる、あなたのキャリア」**
5問の質問に答えるだけで、AIが適正なキャリアロードマップを作成するWebサービス。

- **メインターゲット**: 就活中の学生、転職を考えている社会人、キャリアに悩む20〜30代
- **本ドキュメントの目的**: LP（トップページ）の訴求軸を定めるため、「キャリアに悩む3タイプ」のペルソナを定義する

## ペルソナ1：就活生タイプ「田中 さくら」

### 基本プロフィール
- **年齢 / 性別**: 21歳 / 女性
- **職業 / 立場**: 私立大学 文系学部 3年生
- **居住地**: 地方都市（一人暮らし）
- **ITリテラシー**: 高い（スマホ中心、SNSネイティブ）
- **利用デバイス**: スマートフォン（80%）/ ノートPC（20%）

### 状況・背景
- 周囲が就活を始め、自分も動き出したが「やりたいこと」が明確でない
- 自己分析の本やツールを試したが、結局結論が出ずモヤモヤしている
- 業界研究をしても、自分との接点が分からず手が止まる

### ゴール（実現したいこと）
- 自分の強みや適性を言語化したい
- どの業界・職種を目指せば良いか、当たりをつけたい
- ES・面接で語れる「自分の軸」を持ちたい

### 悩み・障壁（ペインポイント）
- 自己分析に時間をかけても答えが出ない
- 情報が多すぎて、何から手をつければいいか分からない
- 友人と比べて出遅れている焦りがある

### サービスへの期待
- **短時間**で手軽に診断できること（就活で忙しい）
- 結果が**具体的な行動**につながること
- 無料で気軽に試せること

### LPで響く訴求ポイント
- 「5問だけ」「3分で完了」のスピード感
- 「自分の強みが分かる」という自己理解の価値
- スマホで完結する手軽さ

## ペルソナ2：第二新卒・転職検討タイプ「佐藤 健太」

### 基本プロフィール
- **年齢 / 性別**: 26歳 / 男性
- **職業 / 立場**: IT企業 営業職（社会人3年目）
- **居住地**: 東京都（賃貸・一人暮らし）
- **ITリテラシー**: 中〜高
- **利用デバイス**: スマートフォン（60%）/ PC（40%）

### 状況・背景
- 新卒で入った会社に大きな不満はないが、成長実感が薄い
- 「このままで良いのか」という漠然とした不安がある
- 転職サイトに登録したが、求人が多すぎて自分に合うものが分からない

### ゴール（実現したいこと）
- 今の仕事を続けるべきか、転職すべきか判断材料が欲しい
- 自分の市場価値・適性を客観的に知りたい
- 転職するなら、どんな方向性が向いているか知りたい

### 悩み・障壁（ペインポイント）
- 転職エージェントに相談するのはハードルが高い（売り込まれそう）
- 自分の強みが「業務内」のことしか分からず、汎用性が不明
- 失敗したくないので、慎重になりすぎて動けない

### サービスへの期待
- **客観的・中立的**なアドバイスが得られること
- 営業色のない、診断ベースの提案
- すぐに使えて、結果がロードマップとして残ること

### LPで響く訴求ポイント
- 「AIが客観的に診断」という中立性・信頼感
- 「キャリアロードマップ」が手に入る具体性
- エージェントに会う前の「最初の一歩」として最適

## ペルソナ3：キャリア迷子タイプ「鈴木 美咲」

### 基本プロフィール
- **年齢 / 性別**: 31歳 / 女性
- **職業 / 立場**: メーカー 企画職（社会人8年目）
- **居住地**: 政令指定都市（パートナーと同居）
- **ITリテラシー**: 中
- **利用デバイス**: スマートフォン（70%）/ PC（30%）

### 状況・背景
- 仕事は一通りこなせるが、この先のキャリアの方向性が描けない
- 管理職・専門職・転職・ライフイベントなど選択肢が多く迷っている
- 「30代のうちに動くべき」という焦りと、現状維持の安心感の間で揺れている

### ゴール（実現したいこと）
- 中長期のキャリアの方向性を整理したい
- 自分の価値観・優先順位を踏まえた選択をしたい
- 漠然とした不安を「次の一歩」に変えたい

### 悩み・障壁（ペインポイント）
- 選択肢が多く、何を基準に決めればいいか分からない
- キャリア相談に行く時間も気力もない
- 自己流の整理では堂々巡りになる

### サービスへの期待
- 自分の価値観に沿った**ロードマップ**が得られること
- 短時間で頭の整理ができること
- 一度きりでなく、節目で見直せること

### LPで響く訴求ポイント
- 「あなたのキャリアロードマップ」を提示するという中長期視点
- 価値観に寄り添う診断の納得感
- 忙しい人でも「5問・3分」で整理できる手軽さ

---

## 3ペルソナ共通のインサイト

| 観点 | 共通点 |
|------|--------|
| 共通の悩み | 「自分に合うキャリアが分からない」「次の一歩が踏み出せない」 |
| 共通の障壁 | 時間がない / 情報過多 / 相談のハードルが高い |
| 共通の期待 | 手軽・短時間・客観的・具体的な結果 |
| 刺さるキーワード | 「5問」「3分」「AI診断」「ロードマップ」「無料」 |

> **LP設計への示唆**: 全ペルソナに共通する「手軽さ（5問・短時間）」と「具体的な成果（ロードマップ）」を、ヒーローセクションで最優先に訴求する。
