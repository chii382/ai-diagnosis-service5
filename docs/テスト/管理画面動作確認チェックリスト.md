### 事前準備

| 項目 | 内容 |
|------|------|
| 環境変数 | `.env.local` に `AUTH_SECRET` `AUTH_URL` `GOOGLE_CLIENT_ID` `GOOGLE_CLIENT_SECRET` `MONGODB_URI` を設定 |
| 管理者アカウント | `ADMIN_EMAILS` に登録したメール、または MongoDB `users.role` が `"admin"` の Google アカウントを1つ用意 |
| 一般ユーザーアカウント | `role` が `"user"` の Google アカウントを1つ用意 |
| テストデータ | 診断履歴が複数件あると、統計・グラフ・ランキングの確認がしやすい |
| 管理者昇格 | `ADMIN_EMAILS` 設定後は **一度ログアウトして再ログイン** し、JWT に `role: admin` が反映されていること |
| Sentry（任意） | `SENTRY_DSN` / `NEXT_PUBLIC_SENTRY_DSN` を設定すると、エラーログ画面の Sentry 連携表示が「有効」になる |
