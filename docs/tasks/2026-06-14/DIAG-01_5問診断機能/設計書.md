# DIAG-01 会員限定AIキャリア診断機能

## 1. 概要

| 項目 | 内容 |
|------|------|
| 対象 | 会員限定の5問キャリア診断 + AI分析 + 結果CRUD |
| AI | Claude Haiku 4.5（`claude-haiku-4-5-20251001`） |
| DB | MongoDB `diagnoses` コレクション（ネイティブドライバー） |
| 認証 | `/diagnosis` 以下は middleware + ページ側で保護 |

## 2. 画面

| パス | 説明 |
|------|------|
| `/diagnosis` | 5問ウィザード |
| `/diagnosis/result?id=` | 診断直後の結果表示 |
| `/diagnosis/history` | 履歴一覧 |
| `/diagnosis/[id]` | 結果詳細 |
| `/diagnosis/[id]/edit` | 結果編集 |

## 3. API

| メソッド | パス | 説明 |
|----------|------|------|
| POST | `/api/diagnosis` | 回答送信 → AI分析 → 保存 |
| GET | `/api/diagnosis` | 履歴一覧 |
| GET | `/api/diagnosis/[id]` | 詳細取得 |
| PUT | `/api/diagnosis/[id]` | 結果更新 |
| DELETE | `/api/diagnosis/[id]` | 削除 |

## 4. Diagnosis モデル（MongoDB）

```typescript
{
  _id: ObjectId,
  userId: ObjectId,
  answers: { q1..q5: string },
  result: { summary, strengths[], recommendedDirections[], advice },
  careerRoadmap: { shortTerm, midTerm, longTerm },
  createdAt: Date,
  updatedAt: Date,
}
```

## 5. 環境変数

```env
ANTHROPIC_API_KEY=
ANTHROPIC_MODEL=claude-haiku-4-5-20251001  # 任意
```
